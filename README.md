# Application for image rotation with minimal loss of quality

## Settings
Settings file: settings.txt

Data:
  x
  y
  sleep (in millis)

Start image: 1.bmp

## Compatibitily

The program is tested on `Windows 10` operating systems and `Ubuntu 18.04`

## Launch:

### Linux
  ./total [create|rotate]

### Windows
  total.exe [create|rotate]
